# ICanPay介绍

ICanPay是一个提供了多个网关支付处理的类库，使用ICanPay可以简化订单的创建、查询跟接收网关返回的支付通知。

ICanPay目前实现了4家支付网关，你可以通过实现抽象基类以支持更多的网关。

目前支持的支付网关有：支付宝(Alipay)、微信支付(WeChat Payment)、财付通(Tenpay)、易宝(Yeepay)。
